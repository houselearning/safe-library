Pre-installation Notes for the x64-based Versions of the Microsoft Windows Server 2003 and Microsoft Windows XP Professional x64 Edition Operating Systems

This pre-installation information pertains to the following operating systems:

  • Microsoft® Windows Server™ 2003, Standard x64 Edition

  • Windows Server 2003, Enterprise x64 Edition

  • Windows Server 2003, Datacenter x64 Edition

  • Windows® XP Professional x64 Edition

Unisys server hardware

If you acquired your server hardware from Unisys Corporation, before you install any of the x64-based versions of Windows Server 2003, it is very important that you contact Unisys to obtain information that is critical to your installation. For contact information, see the Unisys Product Support Web site.

Additional information and resources

At the time of writing, there were no other known pre-installation issues for the x64-based versions of Windows Server 2003 operating systems or Windows XP Professional x64 Edition.

For information about late-breaking issues and ways to address them, see the Microsoft Knowledge Base.

For more information about the x64-based versions of Windows Server 2003 operating systems, see the Windows Server System Web site.

For more information about Windows XP Professional x64 Edition, see the Windows XP Professional x64 Edition Web site.
